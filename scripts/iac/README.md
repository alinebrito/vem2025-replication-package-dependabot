# Infraestrutura em Nuvem

Este diretório armazena todo o código-fonte necessário para reproduzir a infraestrutura de computação em nuvem a utilizada para a execução da mineração de dados.

---

* `/aws`: Este diretório contém um projeto de infraestrutura codificado em [Terraform](https://developer.hashicorp.com/terraform) para o serviço de computação em nuvem [AWS (Amazon Web Services)](https://aws.amazon.com/pt/).
Ademais, foi incluído na raiz do projeto um arquivo `README.md` detalhando todos os passos necessários para executar o código-fonte e
reproduzir o ecossistema de serviços de computação em nuvem utilizados.
